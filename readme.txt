AI-first e-commerce dashboard for WordPress. Manage products, orders, customers, shipping, and multi-channel sales from a single React-powered admin panel.

== Description ==

Blu Store adds a modern e-commerce dashboard to your WordPress admin. Features include:

* Product management with variants, options, and images
* Order tracking and fulfillment
* Customer management
* Shipping configuration
* Multi-channel sales (Google Merchant Center, Meta Commerce, Etsy, TikTok Shop)
* Stripe Connect for payment processing
* WooCommerce bidirectional sync
* Demo data for testing

== Installation ==

1. Upload the `blu-store` folder to `/wp-content/plugins/`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate to the Blu Store menu item in the admin sidebar
